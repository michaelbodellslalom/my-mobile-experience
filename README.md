# Green Leaf Market - Mobile Loyalty App

A mobile-first web application for Green Leaf Market loyalty members to track rewards points, discover personalized offers, locate stores, and view their environmental impact.

![Green Leaf Market](https://img.shields.io/badge/React-18-blue) ![Vite](https://img.shields.io/badge/Vite-8.0-purple) ![Mobile First](https://img.shields.io/badge/Mobile-First-green)

---

## 🎯 Project Overview

**Type:** P200 Capstone - Mobile-First Web Application  
**Framework:** React 18 + Vite  
**Design:** Responsive, thumb-friendly, sustainability-focused  
**Status:** ✅ MVP Complete

### Key Features
- ✅ Points Dashboard with progress tracking
- ✅ Sustainability Impact Widget (plastic saved, CO2 offset)
- ✅ Personalized Offers with horizontal scrolling
- ✅ Store Locator with directions and calling
- ✅ Purchase History with expandable transactions
- ✅ Bottom navigation (thumb-accessible)
- ✅ Mobile-optimized touch interactions

## 🚀 Quick Start

### Installation

\`\`\`bash
# Install dependencies
npm install

# Start development server
npm run dev
\`\`\`

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

\`\`\`bash
npm run build
npm run preview
\`\`\`

---

## 🌐 Deployment to Vercel

### Quick Deploy

\`\`\`bash
# Login to Vercel
npx vercel login

# Deploy to production
npx vercel --prod
\`\`\`

Your app will be live at: \`https://YOUR-PROJECT.vercel.app\`
